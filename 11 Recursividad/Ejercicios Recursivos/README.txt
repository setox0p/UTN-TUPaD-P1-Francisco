# Descripción
Este repositorio contiene la resolución de los ejercicios propuestos sobre recursividad en Python para  Programación I

Se incluyen funciones recursivas para:
- Cálculo de factorial
- Serie de Fibonacci
- Potencia de un número
- Conversión decimal a binario
- Verificación de palíndromos
- Suma de dígitos
- Cálculo de bloques en una pirámide
- Conteo de dígitos en un número

El código está estructurado, comentado y preparado para ser probado fácilmente.

---

## Uso

1. Clonar el repositorio.
2. Ejecutar el archivo `ejercicios_recursivos.py` con Python.
3. Seguir el menú para probar cada ejercicio.
